# Neo-Vim setting

My repository for Neo-vim config

## Requirements

1. Install NeoVim
2. Install prerequiste packages in `setup.sh`
3. Open nvim, run `:PackerSync`

## Debug

1. Lsp is not working (check using `:LspInfo`): try `:LspStart <lsp-module-name>`
